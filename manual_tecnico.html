Requisitos del sistema y cuentas admitidas

Requisitos mínimos del sistema

iOS: iOS 11

Windows: Windows 10 compilación 14393

Android: Android OS 5,0

Web: Última versión de Safari, Internet Explorer 11, Chrome, Edge o Firefox

Mac: MacOS 10,13

Cuentas admitidas

Microsoft To Do es gratis al usar su cuenta de Microsoft, lo que significa Microsoft To Do estará disponible para todos los clientes Microsoft 365 Familia y personales. Tanto las cuentas de Microsoft gratuitas Microsoft 365 como las suscritas se basan en servidores de Exchange Online, por lo que se cumplen los requisitos técnicos que se describen a continuación.

Los ejemplos comunes de cuentas personales de Microsoft son aquellos que usas para iniciar sesión en Skype, Outlook.com o Xbox.

También puede usar una cuenta profesional o educativa, pero tendrá que asegurarse de que sea compatible en primer lugar.

Las siguientes suscripciones de Microsoft 365 incluyen una licencia para hacer, necesaria para usar Microsoft To Do con una cuenta profesional o educativa:

  • Microsoft 365 Empresa básico
  • Microsoft 365 Estándar para empresas
  • Microsoft 365 Business Premium
  • Microsoft 365  F3
  • Office 365 F2-F3
  • Office 365 E1-E5
  • Microsoft 365 E3-E5
  • Office 365 Educación E3
  • Office 365 a1-A5
  • Microsoft 365 A3-A5
  • Exchange Online (plan 1)
  • Exchange Online (plan 2)
  • Conceptos básicos de Exchange Online
  • Quiosco de Exchange Online
  • POP de Exchange Online

Además de una suscripción válida, se necesita una de las siguientes licencias de Exchange Online para que Microsoft To Do pueda sincronizar los datos de usuario:

  • Exchange Online (plan 1)
  • Exchange Online (plan 2)
  • Exchange Essentials
  • Quiosco de Exchange Online
  • Exchange Online (plan 1)
  • POP de Exchange Online
  • Exchange Online (P1)

No se admiten los siguientes tipos de cuenta o suscripciones:

  • Cuentas hospedadas mediante instalaciones locales de Exchange Server
  • Cuentas gubernamentales/suscripciones
  • Aplicaciones de Microsoft 365 para empresas o Aplicaciones de Microsoft 365 para negocios
  • Microsoft 365 Cuentas/suscripciones a Alemania operadas por sistemas T-Mobile
  • Microsoft 365 Cuentas o suscripciones de China operadas por 21Vianet
